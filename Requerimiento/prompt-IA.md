# 🧠 PROMPT OPTIMIZADO — LANDING PAGE VENTA DE LOTES (MOYOCOCHITA)

---

## CONTEXTO  
Se requiere crear una **landing page de alta conversión** para la venta de lotes en la urbanización **Moyocochita**, utilizando como referencia visual las imágenes de propaganda que serán proporcionadas.

La landing debe estar enfocada en **captar leads (consultas por WhatsApp o formulario)** y generar interés inmediato en potenciales compradores.

---

## ROL  
Actúa como un **experto en CRO, UX/UI, copywriting inmobiliario y desarrollo web frontend**, con experiencia en proyectos de venta de terrenos y bienes raíces en Latinoamérica.

---

## OBJETIVO  
Diseñar una landing page optimizada que:

- Genere confianza inmediata  
- Destaque los beneficios del proyecto  
- Motive a la acción (consulta / contacto)  
- Maximice la conversión de visitantes a leads  

---

## INSTRUCCIONES  

1. Analiza las imágenes proporcionadas y extrae:
   - Propuesta de valor
   - Beneficios clave
   - Mensajes de marketing existentes  

2. Construye la landing con las siguientes secciones:

### 1. Hero Section
- Título potente orientado a venta
- Subtítulo persuasivo
- CTA claro (WhatsApp / consulta)

### 2. Beneficios del Proyecto
- Lista clara (ubicación, plusvalía, accesos, etc.)

### 3. Sobre la Urbanización
- Descripción atractiva
- Enfoque en estilo de vida o inversión

### 4. Galería
- Basada en imágenes proporcionadas

### 5. Ubicación
- Descripción estratégica (cercanía, accesos)

### 6. Oferta / Urgencia
- Escasez, promociones, oportunidad

### 7. CTA Final
- Botón fuerte de contacto

---

## REQUISITOS  

### Funcionales:
- Diseño responsive (mobile-first)
- Botón directo a WhatsApp
- Navegación clara y escaneable

### Técnicos:
- HTML5 limpio
- CSS organizado (preferible Tailwind o separado)
- Código listo para uso en producción básica

---

## RESTRICCIONES  

- No inventar datos críticos (precios, servicios, ubicación exacta)  
- Evitar textos genéricos sin sustento  
- No sobrecargar visualmente la página  
- Priorizar conversión sobre diseño estético  

---

## FORMATO DE SALIDA  

1. Estructura de la landing (wireframe en texto)  
2. Copy completo listo para usar  
3. Código HTML + CSS  
4. Recomendaciones de mejora (opcional)  

---

## CRITERIOS DE CALIDAD  

- Nivel profesional (apto para negocio real)  
- Copy orientado a ventas inmobiliarias  
- UX optimizada para conversión  
- Código limpio, reutilizable y mantenible
